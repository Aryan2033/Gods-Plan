2]  Basic Setup (Do this first always)
    

import logging

logging.basicConfig(
    filename="app.log",
    level=logging.INFO,
    format="%(asctime)s - %(levelname)s - %(message)s"
)

what is this 
    - import
    - filename : stores logs into this file
    - level=logging.INFO : logs INFO and above
    - fromat : how logs get stored

3] Log Levels (VERY IMPORTANT)

    INFO	normal steps

    ERROR	something failed
