1] data_folder.mkdir(exist_ok=True) creates the directory referenced by data_folder.
exist_ok=True meaning: if the directory already exists, do nothing instead of raising FileExistsError.
Default without it: mkdir() (or exist_ok=False) raises FileExistsError when the directory exists.
